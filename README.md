# graph_dejkstra
приложение для визуализации графов, динамического расчета расстояний и поиска кратчайшего пути. Проект написан на C++ с использованием фреймворка Qt.

## Основной функционал

![Uploading image.png…]()



*   **Чтение из файла:** Автоматическая загрузка структуры графа при запуске из текстового файла `input.txt`.
*   **Умная геометрия и динамические веса:** Изначально вершины выстраиваются в форме правильного круга. Вес каждого ребра автоматически высчитывается как точное евклидово расстояние (в пикселях) между соединенными вершинами.
*   **Полная интерактивность (Drag & Drop):** Любую вершину можно захватить и переместить мышью. При этом связи графа тянутся за ней, а веса ребер (расстояния) моментально пересчитываются и обновляются на экране.
*   **Навигация по сцене:** Поддерживается приближение и отдаление рабочей области с помощью колесика мыши.
*   **Поиск кратчайшего пути:** В основе логики лежит алгоритм Дейкстры. Пользователю достаточно кликнуть на начальную (Старт) и конечную (Финиш) вершины, после чего программа мгновенно подсветит оптимальный маршрут.
*   **Анимация движения:** После нахождения кратчайшего пути по нему автоматически начинает ехать машинка. Изображение плавно интерполируется между точками и поворачивается капотом в сторону направления движения.
*   **Экспорт данных:** При каждом новом построении маршрута приложение автоматически сохраняет актуальную матрицу смежности графа в файл `matrix.txt`.

##  Формат входного файла (`input.txt`)

Программа ожидает найти файл `input.txt` рядом с исполняемым файлом (в рабочей директории). Файл должен иметь строгую структуру:
1. Первая строка: `N` — количество вершин.
2. Вторая строка: `M` — количество ребер (связей).
3. Следующие `M` строк: пары чисел `u v`, обозначающие наличие ребра между вершинами `u` и `v`.

**Пример `input.txt`:**
```text
5
6
0 1
0 2
1 3
2 3
3 4
1 4
